# Step 1: 지팡이 입수 (5분)

앱 하나 다운받으면 끝. 터미널, 명령어, 코딩 전혀 필요 없어요.

### 설치 방법

| 순서 | 할 일 |
|---|---|
| 1 | [claude.ai/download](https://claude.ai/download) 접속 |
| 2 | 본인 OS에 맞는 버전 다운로드 |
| 3 | 설치 파일 실행 |

| Mac | Windows |
|---|---|
| .dmg → Applications에 드래그 | .exe 실행 → 설치 완료 |

> 💡 비유: 카카오톡 설치하는 것과 똑같아요.

### 로그인

앱 실행 → Claude 계정으로 로그인 → 끝!

> Pro 구독이 안 되어 있다면? claude.ai → Settings → Billing에서 구독 후 다시 로그인.

<details>
<summary>▸ Git 설치하신 분 — 반드시 앱 재시작!</summary>

Git 설치 후 Claude 앱을 재시작하지 않으면 스킬 설치가 안 돼요.

| Mac | Windows |
|---|---|
| Cmd + Q로 종료 → 다시 실행 | 작업관리자 → Claude 종료 → 다시 실행 |

</details>

### 작업 폴더 만들기

앱에서 "Open Project" → 작업할 폴더 지정.

**추천:** `문서 > ai-workspace`

> 💡 비유: 사무실 서랍을 하나 열어서 AI 앞에 놓는 것. AI는 이 서랍 안의 서류만 봐요.

# Step 2: 마법서 쓰기 (CLAUDE.md)

ChatGPT에 매번 "나는 OO과 의사인데..." 하고 설명하는 거, 해보셨죠?

**마법서를 한 번 써두면, 다음부터는 설명 안 해도 돼요.**

> 💡 비유: 진료실 벽에 붙여둔 메모. AI가 들어올 때마다 먼저 읽어요.

### 만드는 법 — 3단계

| 순서 | 할 일 | 주의 |
|---|---|---|
| 1 | 아래 템플릿을 채팅창에 복붙 | 대괄호만 본인 정보로 수정 |
| 2 | **Enter** 키로 전송 | 복붙만 하면 안 됨! |
| 3 | Claude가 "만들까요?" → **Accept** 클릭 | 승인해야 저장됨 |

### 기본 조작법

| 동작 | 방법 |
|---|---|
| 메시지 전송 | Enter |
| 줄바꿈 (전송 아님) | Shift + Enter |
| 파일 승인 | Accept 버튼 |

<details>
<summary>▸ 마법서 템플릿 (복붙용)</summary>

```
CLAUDE.md 파일을 만들어줘. 내용은:

# [내 이름] — [직군]

## 1. 역할과 맥락
- 직군: [예: 피부과 전문의 / 변호사 / 공인회계사]
- 소속: [예: OO피부과 원장 / OO법률사무소]
- 경력: [예: 개업 3년차]
- 주요 업무: [예: 환자 진료, 병원 경영]
- 반복적으로 하는 일: [ xxx ]

## 2. 작업 원칙
- [예: 답변은 결론부터. 근거는 그 다음에]
- [예: 확실하지 않은 내용은 "확인 필요"라고 명시해]

## 3. 승인 필요 작업
- [예: 고객에게 보내는 문서는 반드시 내가 확인 후 발송]

## 4. 보안 가드레일
- [예: 환자 실명, 주민번호 등 개인정보를 저장하지 마]

## 5. 결과물 품질 기준
- [예: 환자 안내문 — 중학생이 읽어도 이해되는 수준]

## 6. 도구와 워크플로우
- [예: 선호 형식: 표 > 글, 짧은 문장 > 긴 설명]
```

</details>

> 💡 **"반복적으로 하는 일"이 가장 중요해요.** "문서 작성" 보다 → "매주 월요일 환자 리뷰 20개 요약 정리" 가 좋은 주문 문구예요.

# Step 3: 첫 주문 실습

마법서를 썼으니, 첫 주문을 외워볼 차례!

### 시킬 업무가 있는 분

| 직군 | 주문 예시 |
|---|---|
| 변호사 | "이번 주 받은 계약서에서 을에게 불리한 조항을 찾아줘" |
| 의사 | "지난달 매출 데이터를 시술별 수익성으로 정리해줘" |
| 회계사 | "부가세 신고 체크리스트를 만들어줘" |
| 피부과 | "최근 피부과 트렌드를 정리해서 블로그 글 초안 써줘" |

### 뭘 시켜야 할지 모르겠는 분

```
"내가 매주 반복하는 업무 3가지를 말해줄 테니,
각각에 대해 체크리스트를 만들어줘"
```

<details>
<summary>▸ 결과가 마음에 안 들면? — 주문 다듬기</summary>

| 이렇게 말하면 | 효과 |
|---|---|
| "좀 더 간결하게" | 분량 축소 |
| "표로 정리해줘" | 구조화 |
| "이 부분을 더 자세히" | 특정 부분 확장 |
| "전문 용어를 쉽게 풀어줘" | 난이도 조정 |
| "실제 사례를 넣어줘" | 구체성 추가 |

> It's leviOsa, not levioSA — 주문 문구를 다듬을수록 결과가 좋아져요 🪶

</details>

## 주문이 안 먹힐 때

<details>
<summary>▸ 트러블슈팅 표 펼치기</summary>

| 문제 | 해결 |
|---|---|
| 앱 설치 안 됨 (Mac) | 시스템 설정 → 보안 → "확인 없이 열기" |
| 앱 설치 안 됨 (Windows) | "추가 정보" → "실행" 클릭 |
| 로그인 후 반응 없음 | 앱 종료 → 재실행 |
| Pro 구독 안 됨 | claude.ai → Settings → Billing |
| "사용량 초과" | 잠시 기다렸다 재시도 |
| CLAUDE.md 안 읽힘 | 폴더 최상위에 파일 있는지 확인 |
| Git 후 스킬 안 됨 | 앱 완전 종료 → 재실행 |
| 복붙 후 반응 없음 | Enter 눌렀는지 확인! |

</details>
