오늘은 입학 첫날이에요. 지팡이를 잡는 것부터 시작할게요.

머글 시절엔 이걸 "소프트웨어 설치"라고 불렀어요. 하지만 여기서는 **지팡이를 고르는 시간**이에요. 지팡이가 없으면 주문을 외울 수 없으니까요.

오늘의 지팡이는 **Claude Code** 데스크톱 앱이에요.

# 지팡이 입수 (5분)

터미널? 명령어? 코딩? 전부 필요 없어요. 앱 하나 다운받으면 끝이에요.

## Step 1: Claude Code 데스크톱 앱 다운로드

1. [claude.ai/download](https://claude.ai/download) 접속
2. 본인 OS에 맞는 버전 다운로드
3. 설치 파일 실행

| Mac | Windows |
|---|---|
| .dmg 파일 열기 → Applications에 드래그 | .exe 파일 실행 → 설치 완료 |

> 카카오톡 설치하는 것과 똑같아요. 앱 하나 깔면 끝. 터미널, 명령어, 코딩 전혀 필요 없습니다.

- 오늘 앱 실습만 하시는 분은 추가 설치가 필요 없어요.
- 추후 터미널 기반 Claude Code를 더 깊게 쓰고 싶다면, 별도로 Git 설치가 필요할 수 있어요.

## Step 2: 로그인

1. 앱 실행
2. Claude 계정으로 로그인 (claude.ai와 같은 계정)
3. 끝!

> Pro 구독이 안 되어 있다면? claude.ai → Settings → Billing에서 Pro 구독 후 다시 로그인하세요.

### Git 설치하신 분 — 반드시 앱 재시작!

> Git을 설치한 뒤에는 Claude 앱을 완전히 종료하고 다시 열어야 해요. 재시작하지 않으면 스킬 설치 등이 제대로 안 돼요. 주문 역효과가 날 수 있어요 😅

| Mac | Windows |
|---|---|
| Cmd + Q로 앱 종료 → 다시 실행 | 작업관리자(Ctrl+Shift+Esc) → Claude 선택 → "작업 끝내기" → 다시 실행 |

## Step 3: 작업 폴더 만들기

앱에서 "Open Project" 또는 "New Project"를 눌러 작업할 폴더를 지정해요.

추천: **문서 폴더에 `ai-workspace` 같은 폴더를 하나 만들어두세요.**

| Mac | Windows |
|---|---|
| 문서 > ai-workspace | 문서 > ai-workspace |

> 왜 폴더가 필요한가요? Claude Code는 폴더 단위로 작업해요. 이 폴더 안에 CLAUDE.md 파일을 만들면, AI가 이 폴더에서 작업할 때마다 여러분의 맥락을 기억합니다. 마법서를 서재에 꽂아두는 것과 같아요.

# 나만의 마법서(CLAUDE.md) 쓰기

CLAUDE.md는 Claude Code에게 "나는 이런 사람이고, 이런 일을 한다"라고 알려주는 파일이에요.

그냥 자기소개가 아니에요. **AI가 나와 일할 때 지켜야 할 규칙**이에요.

md는 Markdown의 줄임말인데요, 글을 쉽게 정리하고 구조화할 수 있는 문서 형식이에요. AI는 이 형식을 아주 잘 이해해요.

ChatGPT에 매번 "나는 OO과 의사인데..." 하고 설명하는 거, 해보셨죠?

**이 마법서를 한 번 써두면, 다음부터는 설명 안 해도 돼요.**

교과서 1페이지에 나와 있어요 — 매번 자기소개하는 건 비효율적이에요.

## 만드는 방법

1. Claude Code 앱에서 아까 만든 폴더를 연다 (문서 > ai-workspace)
2. 채팅창에 아래 템플릿을 복붙하고 **대괄호만 본인 정보로 바꾼다**
3. **Enter(엔터)** 키를 눌러 전송한다
4. Claude가 "이 파일을 만들까요?" 하고 물어봅니다 → **"Accept" 버튼을 클릭**하면 끝!

### 꼭 기억하세요!

| 동작 | 방법 | 설명 |
|---|---|---|
| 메시지 전송 | Enter 키 | 채팅창에 입력 후 보내기 |
| 파일 생성/수정 승인 | "Accept" 버튼 클릭 | Claude가 보여주는 변경사항을 승인 |
| 줄바꿈 (전송 아님) | Shift + Enter | 긴 글 쓸 때 줄바꿈만 |

> 복붙만 하고 Enter를 안 누르면 아무 일도 안 일어나요! 그리고 Claude가 파일을 만들겠다고 보여주면, "Accept"를 눌러야 실제로 저장돼요.

파일을 직접 만들 필요 없어요. Claude Code에게 "만들어줘"라고 시키면 됩니다.

## 마법서 템플릿

아래 내용을 복붙해서 Claude Code에게 시키세요. **대괄호 [ ] 안만 본인 정보로 바꾸면 돼요.**

```
CLAUDE.md 파일을 만들어줘. 내용은:

# [내 이름] — [직군]

## 1. 역할과 맥락
- 직군: [예: 피부과 전문의 / 변호사 / 공인회계사]
- 소속: [예: OO피부과 원장 / OO법률사무소 / OO회계법인]
- 경력: [예: 개업 3년차 / 근무 5년차]
- 주요 업무: [예: 환자 진료, 병원 경영 / 기업자문, 계약서 검토]
- 반복적으로 하는 일: [ xxx ]

## 2. 작업 원칙
- [예: 답변은 결론부터. 근거는 그 다음에]
- [예: 숫자가 들어간 답변은 반드시 출처나 근거를 포함해]
- [예: 확실하지 않은 내용은 "확인 필요"라고 명시해]

## 3. 승인 필요 작업 (AI가 알아서 하면 안 되는 것)
- [예: 고객/환자에게 보내는 문서는 반드시 내가 확인 후 발송]
- [예: 금액이 포함된 견적/제안서는 최종 확인 필수]

## 4. 보안 가드레일 (절대 하면 안 되는 것)
- [예: 환자 실명, 주민번호 등 개인정보를 저장하거나 출력하지 마]
- [예: 고객사 계약 조건, 단가, 예산을 기억하지 마]

## 5. 결과물 품질 기준
- [예: 법률의견서 — 쟁점별 정리, 근거 조문 포함, 3페이지 이내]
- [예: 환자 안내문 — 전문용어 없이, 중학생이 읽어도 이해되는 수준]
- [예: 이메일 — 3문장 이내, 요청사항 명확히]

## 6. 도구와 워크플로우
- [예: 주로 쓰는 도구: Excel, 한글, 네이버 플레이스]
- [예: 선호 형식: 표 > 글, 짧은 문장 > 긴 설명]
```

> "반복적으로 하는 일"이 가장 중요해요. 여기가 AI에게 시킬 업무 목록이에요. 구체적일수록 좋아요 — "문서 작성" 보다 **"매주 월요일 환자 리뷰 20개 요약 정리"** 가 훨씬 좋은 주문 문구예요.

# 첫 주문 실습: 내 업무를 AI에게 시켜보기

마법서를 썼으니, 이제 첫 주문을 외워볼 차례예요.

CLAUDE.md를 작성한 상태이므로, AI가 여러분의 직군과 업무를 이미 이해하고 있어요. 매번 설명할 필요 없어요 — 마법서의 힘이에요.

## 시킬 업무가 있는 분

설문에서 "AI로 해결하고 싶은 업무"라고 쓰신 그대로 입력하세요. 예시:

```
"이번 주 받은 계약서에서 을에게 불리한 조항을 찾아줘"
"지난달 매출 데이터를 분석해서 시술별 수익성을 정리해줘"
"부가세 신고 체크리스트를 만들어줘"
"최근 피부과 트렌드를 정리해서 블로그 글 초안 써줘"
```

## 뭘 시켜야 할지 모르겠는 분

아래를 복붙하세요:

```
"내가 매주 반복하는 업무 3가지를 말해줄 테니,
각각에 대해 체크리스트를 만들어줘"
```

## 결과가 마음에 안 들면?

주문 문구를 다듬으면 돼요:

```
"좀 더 간결하게"
"표로 정리해줘"
"이 부분을 더 자세히"
"전문 용어를 쉽게 풀어줘"
"실제 사례를 넣어줘"
```

> 결과물이 완벽하지 않아도 괜찮아요. "아, 이 정도면 내 업무에 쓸 수 있겠다"가 오늘의 목표예요. 주문은 쓸수록 정교해져요. It's leviOsa, not levioSA — 발음이 중요하듯이, 주문 문구도 다듬을수록 좋아져요 🪶✨

# 주문이 안 먹힐 때 (트러블슈팅)

| 문제 | 해결 방법 |
|---|---|
| 앱이 설치 안 됨 (Mac) | 시스템 설정 → 개인정보 보호 및 보안 → "확인 없이 열기" |
| 앱이 설치 안 됨 (Windows) | "추가 정보" → "실행" 클릭 |
| 로그인 후 반응 없음 | 앱 종료 후 다시 실행 |
| Pro 구독 안 되어 있음 | claude.ai → Settings → Billing에서 Pro 구독 |
| "사용량 초과" 메시지 | 잠시 기다렸다가 다시 시도 (Pro는 시간당 제한 있음) |
| CLAUDE.md가 안 읽히는 것 같음 | 프로젝트 폴더 최상위에 파일이 있는지 확인 |
| Git 설치 후 스킬이 안 됨 | Claude 앱을 완전히 종료 후 다시 실행 |
| 복붙했는데 아무 반응 없음 | Enter 키를 눌렀는지 확인! 붙여넣기만 하면 전송 안 돼요 |

> Reparo. 주문이 안 먹히면 당황하지 마세요. 위 표에서 증상을 찾아보세요. 대부분 앱 재시작으로 해결돼요 🔧✨
