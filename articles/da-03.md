# DA 수업 #3 — 어떤 지팡이를 골라야 할까 🪄

> AI 도구(지팡이)가 너무 많아요. ChatGPT, Grok, Gemini, Claude...
> 다 써보고 정리했어요. 교과서에 나와 있거든요 📖

---

## 결론부터

**하나만 쓸 거면 Claude를 추천해요.**

| 이유 | 설명 |
|---|---|
| 긴 문서를 끝까지 읽어요 | 100페이지 PDF를 올려도 앞부분만 대충 읽고 넘기지 않아요 |
| 지시한 대로 해요 | "표로 정리해줘"라고 하면 진짜 표로 나와요 |
| 깊은 분석에 강해요 | 계약서 조항 검토, 데이터 패턴 분석 같은 "생각이 필요한 작업"에서 차이 |

> 💡 비유: 올리밴더 가게에 지팡이가 수십 개 있지만, 마법사에게 맞는 지팡이는 정해져 있어요. 전문가 업무에는 Claude가 잘 맞아요.

## 그러면 다른 지팡이는 언제 쓰나요?

| 상황 | 추천 도구 | 이유 |
|---|---|---|
| "최신 뉴스를 검색해서 정리해줘" | ChatGPT 또는 Grok | 실시간 웹 검색이 돼요 |
| "이 내용으로 이미지 만들어줘" | ChatGPT (DALL-E) | 이미지 생성 기능 |
| "X(트위터)에서 이 주제 반응 찾아줘" | Grok | X 데이터 직접 연동 |
| "구글 드라이브 문서를 분석해줘" | Gemini | 구글 생태계(Google Workspace) 연동 |

<details>
<summary>▸ 그래서 여러 개 써야 하나요?</summary>

아니요. **90%의 전문가 업무는 Claude 하나로 충분해요.**

나머지 10% — 실시간 검색, 이미지 생성 — 가 필요할 때만 다른 도구를 보조로 쓰면 돼요.

처음부터 여러 개 쓰면 오히려 헷갈려요. 하나를 잘 쓰는 게 먼저예요.

> 론이 한 번에 빗자루 세 개를 타려다 떨어진 것처럼요 ㅎㅎ

</details>

## Claude를 선택하는 진짜 이유 — Claude Code

| 일반 AI (채팅) | Claude Code (지팡이) |
|---|---|
| 대화창에 질문 → 답변 복사 | 내 컴퓨터 파일을 직접 읽고 수정 |
| 매번 복사-붙여넣기 | 폴더 안에서 바로 작업 |
| 맥락을 매번 설명 | CLAUDE.md(마법서)로 맥락 자동 로드 |
| 한 번 쓰고 끝 | 스킬(저장된 주문)로 반복 실행 |

> 💡 비유: 다른 AI는 전화 상담이에요. Claude Code는 옆자리에 앉은 동료예요. 파일을 직접 보고, 고치고, 정리해줘요.
