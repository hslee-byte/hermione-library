# DA 수업 #4 — 마법이 진짜 되는 순간 ✨

> "AI가 좋다는 건 알겠는데, 구체적으로 뭘 할 수 있는 건가요?"
> 직접 해보고, 시간을 재봤어요. 아래 6가지는 진짜 돼요.

---

## 오늘의 목적

기능 설명이 아니에요. **"이런 게 된다고?!"** 하는 순간을 보여드릴게요.

각 사례마다 "사람이 직접 하면" vs "AI(주문)에게 시키면"을 비교했어요.

---

## 사례 1: 100페이지 PDF 요약

| | 머글 방식(수작업) | 주문(AI) |
|---|---|---|
| 소요 시간 | 2~3시간 | 5분 |
| 결과물 | 머릿속 대충 정리 | 핵심 논점 + 데이터 + 표 |
| 놓치는 부분 | 뒷부분 갈수록 집중력 저하 | 끝까지 균일하게 읽음 |

> 💡 비유: 도서관에서 책 한 권을 밤새 읽는 것(머글 방식) vs 속독 주문(AI)으로 5분 만에 핵심 파악

<details>
<summary>▸ 실제 주문(프롬프트) 예시</summary>

```
이 PDF를 읽고 아래 형식으로 요약해줘:
1. 핵심 논점 3가지
2. 주요 데이터/숫자 (표로)
3. 내일 회의에서 언급할 포인트 2개
```

</details>

---

## 사례 2: 계약서 위험 조항 검토

| | 머글 방식 | 주문 |
|---|---|---|
| 소요 시간 | 1~2시간 | 30초 |
| 검토 범위 | 집중력에 따라 다름 | 전체 조항 균일 검토 |
| 결과 형식 | 머릿속 or 메모장 | 조항번호 + 리스크 등급 + 이유 (표) |

> 💡 변호사라면: "을에게 불리한 조항을 리스크 높은 순으로 정리해줘" — 이 한 마디예요.

---

## 사례 3: 매출 데이터 분석

| | 머글 방식 | 주문 |
|---|---|---|
| 소요 시간 | 1시간+ (엑셀 수식) | 3분 |
| 과정 | SUMIF, 피벗테이블, 수식 오류 디버깅 | "시술별로 나눠서 전월 대비 표로" |
| 결과 | 숫자는 나오는데 해석은 별도 | 숫자 + 해석 + 이상치 지적 |

> 💡 회계사/의사라면: 함수를 외우는 게 아니라 말로 시키는 거예요. Accio 분석 결과!

---

## 사례 4: 환자/고객 안내문 작성

| | 머글 방식 | 주문 |
|---|---|---|
| 소요 시간 | 30분 | 2분 |
| 문제 | 전문 용어가 빠지지 않음 | "중학생이 읽어도 이해되게" 지시 가능 |
| 톤 조절 | 직접 여러 번 수정 | "친근하게 / 공식적으로" 한 마디 |

> 💡 의사라면: "이 시술에 대해 환자 안내문을 써줘. 전문 용어 없이, 불안하지 않은 톤으로"

---

## 사례 5: 이메일 초안

| | 머글 방식 | 주문 |
|---|---|---|
| 소요 시간 | 15분 (고민 포함) | 1분 |
| 고민 포인트 | 톤, 길이, 뭐부터 쓸지 | 주문 한 줄이면 초안 완성 |
| 수정 | 처음부터 다시 | "좀 더 간결하게" 한 마디 |

---

## 사례 6: 주간 보고서

| | 머글 방식 | 주문 |
|---|---|---|
| 소요 시간 | 40분 | 5분 |
| 양식 | 매번 빈 문서에서 시작 | CLAUDE.md(마법서)에 양식 저장 → 자동 적용 |
| 일관성 | 쓰는 사람 컨디션에 따라 다름 | 항상 같은 구조 |

<details>
<summary>▸ 전체 비교 한눈에</summary>

| 업무 | 머글 방식 | 주문 | 절약 시간 |
|---|---|---|---|
| PDF 100p 요약 | 2~3시간 | 5분 | **97%** |
| 계약서 검토 | 1~2시간 | 30초 | **99%** |
| 매출 분석 | 1시간+ | 3분 | **95%** |
| 안내문 작성 | 30분 | 2분 | **93%** |
| 이메일 초안 | 15분 | 1분 | **93%** |
| 주간 보고서 | 40분 | 5분 | **88%** |

</details>

## 주의 — 주문의 한계

| AI가 잘하는 것 | AI가 못하는 것 (마법사의 영역) |
|---|---|
| 초안 작성 | **최종 판단** |
| 정보 정리/요약 | **사실 확인 (꼭 검증!)** |
| 패턴 분석 | **대면 소통** |
| 형식 변환 | **윤리적 결정** |

> 주문은 후보를 뽑아줄 뿐. 판단은 마법사의 영역이에요. 확인 안 하면 잠이 안 와요 📖
