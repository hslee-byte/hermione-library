# DA 수업 #6 — 내 서류를 AI에게 읽히다 📄

> 5편까지 주문 문구(프롬프트)를 다듬는 법을 배웠어요.
> 그런데 진짜 업무는 **내 서류를 AI에게 직접 건네는 것**에서 시작돼요. 오늘은 파일을 넣는 법을 배울 거예요!

---

## 핵심 비유

> 💡 옆자리 동료에게 "이 서류 좀 봐줘"하고 건네는 거예요. AI한테도 똑같이 파일을 건네면 돼요. 파일 업로드(서류 전달)라고 해요.

---

## 파일 넣는 법 2가지

| | 드래그앤드롭 (직접 건네기) | 작업 폴더에 미리 넣기 (서랍에 넣어두기) |
|---|---|---|
| **방법** | 파일을 대화창에 끌어다 놓기 | 프로젝트 폴더에 파일을 미리 복사 |
| **언제 쓰나** | 지금 당장 이 파일 하나만 볼 때 | 여러 파일을 반복적으로 참조할 때 |
| **비유** | 서류를 손으로 건네는 것 | 서랍에 넣어두고 "저기 있는 거 봐" |
| **장점** | 간편, 즉시 | 매번 안 건네도 됨 |
| **단점** | 매번 다시 건네야 함 | 폴더 정리 필요 |

> 💡 비유: 드래그앤드롭은 "이거 봐줘"하고 직접 건네는 것, 폴더는 서랍에 넣어두고 "거기 꺼내서 봐"하는 거예요.

---

## 지원하는 파일 형식

| 파일 형식 | 확장자 | AI가 할 수 있는 것 |
|---|---|---|
| PDF | .pdf | 읽기, 요약, 조항 추출, 비교 |
| 엑셀 | .xlsx, .csv | 데이터 분석, 표 정리, 계산 |
| 워드 | .docx | 수정, 요약, 톤 변경 |
| 텍스트 | .txt, .md | 읽기, 편집, 구조화 |
| 이미지 | .png, .jpg | 내용 인식, 텍스트 추출 |

<details>
<summary>▸ 파일 크기 & 주의사항</summary>

| 주의 상황 | 해결법 |
|---|---|
| PDF가 100페이지 이상 | 필요한 부분만 발췌해서 넣기 |
| 스캔 PDF (이미지로 된 문서) | OCR 처리된 파일이면 OK, 아니면 텍스트 추출이 어려움 |
| 엑셀 시트가 여러 개 | "Sheet2의 B열을 봐줘"처럼 시트와 범위 지정 |
| 파일이 5개 이상 | 한 세션 한 업무 원칙 — 관련 파일만 넣기 |

</details>

---

## 실전: 파일 + 주문 문구 조합

파일만 넣으면 AI가 "뭘 하라는 건지" 몰라요. **파일 + 주문 문구**가 한 세트예요.

| ❌ 파일만 넣고 | ✅ 파일 + 주문 문구 |
|---|---|
| [계약서.pdf 첨부] "봐줘" | [계약서.pdf 첨부] "을에게 불리한 조항을 찾아서 조항 번호별로 리스크를 표로 정리해줘" |
| [매출.xlsx 첨부] "분석해줘" | [매출.xlsx 첨부] "시술 항목별 3월 매출을 전월 대비 증감률과 함께 표로 보여줘" |
| [처방전.pdf 첨부] "확인해줘" | [처방전.pdf 첨부] "이 처방에서 약물 상호작용 위험이 있는 조합을 찾아줘" |

> 💡 비유: 서류를 건네면서 "뭘 봐야 하는지"를 같이 말해줘야 해요. 동료한테도 그렇게 하잖아요!

## 작업 흐름 정리

```
1. 폴더를 연다 (서랍을 열어요)
2. 파일을 넣는다 (서류를 건네요)
   ├── 드래그앤드롭: 지금 바로 건네기
   └── 폴더에 미리 넣기: 서랍에 넣어두기
3. 주문 문구를 쓴다 ("이 서류에서 ~을 찾아줘")
4. 결과를 받고, 다듬는다
```
