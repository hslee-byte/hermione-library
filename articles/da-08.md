# DA 수업 #8 — Accio 피벗테이블! 숫자 업무를 말로 📊

> 엑셀 수식을 외우느라 고생한 적 있죠?
> 이제 수식을 외울 필요 없어요. **말로 시키면** AI가 해줘요. 머글 수학(엑셀 함수) 대신 주문(자연어)을 써봐요!

---

## 핵심 비유

> 💡 머글(일반인)은 복잡한 도구(수식)를 손으로 조작해요. 마법사(AI 사용자)는 주문(자연어)을 외우면 결과가 나와요. "Accio 피벗테이블!"이면 끝이에요.

---

## 머글 방식 vs 마법사 방식

| 업무 | 머글 방식 (수식) | 마법사 방식 (주문) |
|---|---|---|
| 다른 시트에서 값 가져오기 | `=VLOOKUP(A2,Sheet2!A:B,2,0)` | "Sheet2에서 고객명 기준으로 매출액을 가져와줘" |
| 항목별 합계 | `=SUMIF(B:B,"시술A",C:C)` | "시술 항목별 매출 합계를 구해줘" |
| 피벗테이블 | 삽입 → 피벗테이블 → 행/열/값 드래그 | "월별, 항목별로 매출을 피벗테이블로 만들어줘" |
| 조건부 서식 | 홈 → 조건부서식 → 규칙 관리 | "전월 대비 감소한 셀을 빨간색으로 표시해줘" |
| 중복 찾기 | `=COUNTIF(A:A,A2)>1` | "이 목록에서 중복된 항목을 찾아줘" |

> 수식을 몰라도 **원하는 결과를 말로 설명**하면 돼요. AI가 수식을 만들어주거나, 직접 계산해줘요.

---

## 실전 예시 3가지

### 예시 1: 매출 분석

```
[매출.xlsx 첨부]

이 매출 데이터를 분석해줘:
1. 시술 항목별 매출 합계를 표로
2. 전월 대비 증감률을 계산해서 옆에 붙여줘
3. 매출 상위 5개 항목을 하이라이트
```

### 예시 2: 경비 분류

```
[경비.csv 첨부]

이 경비 내역을 분류해줘:
1. 항목별(교통비, 식비, 사무용품 등)로 자동 분류
2. 분류별 합계를 표로
3. 비정상적으로 큰 금액이 있으면 표시
```

### 예시 3: 전년 비교

```
[2025매출.xlsx, 2024매출.xlsx 첨부]

두 파일을 비교해줘:
1. 월별 매출을 나란히 표로
2. 전년 대비 증감률 계산
3. 가장 큰 변동이 있는 항목 3개를 강조
```

---

## 주문의 핵심 구조

좋은 숫자 주문에는 3가지가 있어요:

| 요소 | 설명 | 예시 |
|---|---|---|
| **데이터** | 어떤 파일/시트/범위 | "Sheet1의 A~D열" |
| **계산** | 무엇을 구할지 | "항목별 합계, 전월 대비 증감률" |
| **출력** | 어떤 형태로 보여줄지 | "표로, 상위 5개 강조" |

> 💡 비유: 마법 주문도 지팡이(도구) + 주문어(의도) + 동작(형태)이 맞아야 효과가 나오듯, 숫자 주문도 데이터 + 계산 + 출력이 한 세트예요.

## 결과가 이상하면?

| 상황 | 해결 주문 |
|---|---|
| 숫자가 안 맞는 것 같을 때 | "계산 과정을 단계별로 보여줘" |
| 빈 칸이 있어서 계산이 이상할 때 | "빈 칸은 0으로 처리하고 다시 계산해줘" |
| 원하는 형식이 아닐 때 | "이걸 피벗테이블 형태로 다시 정리해줘" |
