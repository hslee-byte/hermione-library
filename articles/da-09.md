# DA 수업 #9 — Revelio! 데이터를 그림으로 📈

> 8편에서 숫자를 말로 다루는 법을 배웠어요.
> 그런데 숫자만 잔뜩 있으면 아무도 안 봐요. **그림(차트)으로 바꾸면 1초 만에 이해**돼요!

---

## 핵심 비유

> 💡 Revelio! 는 숨겨진 것을 드러내는 주문이에요. 숫자 속에 숨은 패턴(시각화, Visualization)을 드러내는 마법이죠. 표에서는 안 보이던 추세가 차트에서는 한눈에!

---

## 언제 어떤 차트를 쓰나?

| 차트 유형 | 언제 쓰나 | 예시 |
|---|---|---|
| 막대 차트 (Bar) | 항목 간 크기 비교 | 시술별 매출 비교, 부서별 인원 |
| 파이 차트 (Pie) | 전체 대비 비율 | 질환별 환자 비율, 경비 항목 비율 |
| 꺾은선 차트 (Line) | 시간에 따른 변화 | 월별 매출 추이, 수치 변화 추적 |
| 가로 막대 (Horizontal Bar) | 항목이 많을 때 비교 | 20개 거래처 매출 순위 |
| 누적 막대 (Stacked Bar) | 구성 비율 + 크기 동시 비교 | 월별 시술 유형 구성 |
| 히트맵 (Heatmap) | 두 축의 교차 패턴 | 요일×시간대 방문객, 월×항목 매출 |

<details>
<summary>▸ 차트 선택이 헷갈릴 때</summary>

AI에게 이렇게 시키면 돼요:

```
이 데이터를 가장 잘 보여줄 수 있는 차트 유형을 추천해줘.
왜 그 차트가 적합한지도 설명해줘.
```

AI가 데이터 특성을 보고 알맞은 차트를 추천해줘요!

</details>

---

## 주문 예시 3가지

### 예시 1: 막대 차트 — 항목별 비교

```
[매출.xlsx 첨부]

시술 항목별 3월 매출을 막대 차트로 그려줘.
매출 높은 순으로 정렬하고, 상위 3개는 다른 색으로 표시해줘.
```

### 예시 2: 파이 차트 — 비율 파악

```
[경비.csv 첨부]

이번 분기 경비를 항목별 비율로 파이 차트를 그려줘.
5% 미만은 "기타"로 묶어줘.
```

### 예시 3: 꺾은선 차트 — 추이 분석

```
[월별매출.xlsx 첨부]

최근 12개월 매출 추이를 꺾은선 차트로 그려줘.
전년 동기 데이터도 같이 넣어서 비교할 수 있게 해줘.
```

## DA#8에서 이어지는 흐름

| 단계 | DA# | 하는 일 |
|---|---|---|
| 1단계 | DA#6 | 파일을 AI에게 건넨다 |
| 2단계 | DA#8 | 숫자를 말로 분석한다 |
| 3단계 | **DA#9** | 분석 결과를 그림으로 바꾼다 |

```
[매출.xlsx 첨부]

1. 시술 항목별 매출 합계를 구해줘 (DA#8)
2. 전월 대비 증감률을 계산해줘 (DA#8)
3. 결과를 막대 차트로 그려줘, 증감은 색으로 구분 (DA#9)
```

> 💡 비유: DA#8이 투명 망토(데이터를 꿰뚫어 보기)라면, DA#9는 Revelio 주문(눈에 보이게 드러내기)이에요.
