# DA 수업 #12 — 폴더별 마법서 📂

> DA 수업 #2에서 폴더 = 방이라고 배웠어요.
> 오늘은 **각 방마다 다른 브리핑 메모(Project CLAUDE.md)를 넣는 법**을 알려드릴게요.
> 이게 되면, Claude가 방에 들어서는 순간 "이 방에서 뭘 해야 하는지" 바로 알아요.

---

## 왜 분리해야 하나요?

### 섞어서 쓸 때 vs 분리했을 때

| | 🚫 한 폴더에 다 섞기 | ✅ 폴더별로 분리 |
|---|---|---|
| **맥락** | "이 계약서가 어느 클라이언트 건이지?" | 폴더 열면 자동으로 맥락 설정 |
| **톤** | 환자 안내문에 법률 용어가 섞임 | 폴더별 톤 자동 적용 |
| **규칙** | 매번 "너는 ~야" 반복 | 한 번 적어두면 영구 적용 |
| **실수** | A 업무 규칙이 B에 적용되는 혼선 | 칸막이(폴더)가 막아줌 |
| **비유** | 서랍 하나에 모든 서류를 넣는 것 | 서랍별로 라벨 + 브리핑 메모 |

> 💡 비유: 호그와트에서 변신술 교실에 들어가면 변신술 교과서가 열리고,
> 약초학 교실에 들어가면 약초학 교과서가 열리잖아요?
> 폴더 = 교실, Project CLAUDE.md = 그 교실의 교과서예요 📂

---

## 구조 — 이렇게 생겼어요

```
내 작업 폴더/
├── 계약서-검토/           ← 폴더 1 (교실 1)
│   ├── CLAUDE.md          ← 이 폴더 전용 마법서
│   └── (계약서 파일들)
│
├── 환자-안내문/           ← 폴더 2 (교실 2)
│   ├── CLAUDE.md          ← 이 폴더 전용 마법서
│   └── (안내문 관련 파일들)
│
└── 매출-분석/             ← 폴더 3 (교실 3)
    ├── CLAUDE.md          ← 이 폴더 전용 마법서
    └── (엑셀, CSV 파일들)
```

> 각 CLAUDE.md는 **그 폴더에서만** 적용돼요. 다른 폴더에 영향 없어요.

---

## Project CLAUDE.md에 뭘 적나요?

| 항목 | 설명 | 예시 |
|---|---|---|
| **역할** | 이 폴더에서 AI가 뭘 하는 사람인지 | "너는 10년차 기업법 변호사야" |
| **규칙** | 이 업무에서 반드시 지켜야 할 것 | "법령 인용 시 조문 번호 필수" |
| **형식** | 결과물 형식 지정 | "항상 표로 정리, 리스크 등급 포함" |
| **참고 자료** | 항상 참고할 배경 정보 | "우리 사무소 계약서 검토 기준은..." |
| **하지 말 것** | 실수 방지 규칙 | "의학적 진단을 내리지 마" |

## 글로벌 마법서 vs 프로젝트 마법서

| | 글로벌 CLAUDE.md | Project CLAUDE.md |
|---|---|---|
| **위치** | `~/.claude/CLAUDE.md` | 각 폴더 안의 `CLAUDE.md` |
| **적용 범위** | 어디서든 항상 적용 | 그 폴더에서만 적용 |
| **용도** | 내 공통 스타일, 기본 규칙 | 업무별 전문 규칙 |
| **비유** | 교복 (어디서든 입는 것) | 교실별 실험복 (그 교실에서만) |

> 💡 둘 다 쓰면 합쳐져요. 글로벌 = 기본 성격, 프로젝트 = 전문 지식.
> 마치 교복 위에 실험복을 입는 것처럼요 🧪
