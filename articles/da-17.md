# DA 수업 #17 — AI가 내 지팡이를 직접 쓴다 🖥️

> 지금까지 우리는 AI에게 말로 시켰어요. "이거 해줘", "저거 정리해줘."
> 그런데 이제 AI가 **화면을 직접 보고, 마우스를 움직이고, 클릭**해요.

---

## 기존 방식 vs Computer Use — 뭐가 다른가?

| | 기존 방식 (대화형) | Computer Use (화면 제어형) |
|---|---|---|
| AI가 보는 것 | 내가 복붙해 준 텍스트 | **내 화면 전체** |
| AI가 하는 것 | 텍스트로 답변 | **직접 클릭, 타이핑, 스크롤** |
| 비유 | 전화로 비서에게 지시 | 비서가 내 자리에 앉아서 직접 실행 |
| 세계관 비유 | 부엉이에게 편지 보내기 (Owl Post) | **마법사가 내 지팡이를 빌려서 직접 주문** |

> 💡 Computer Use(컴퓨터 사용)란: AI가 사람처럼 화면을 보고(스크린샷 인식), 마우스와 키보드를 조작하는 기술이에요.

---

## 체험기 — "이거 진짜 움직이는 거야?"

처음 Computer Use를 봤을 때 솔직히 소름이 돋았어요.

> 화면에서 AI가 브라우저를 열고, 검색창에 타이핑하고, 결과를 읽고, 필요한 부분을 복사해서 문서에 붙여넣더라고요. 제가 한 건 "이거 해줘" 한 마디뿐이었어요.

마치 투명 인간이 내 옆에 앉아서 컴퓨터를 쓰는 느낌.

---

## 기존 대화형과 나란히 비교

### 예시: "경쟁사 가격표를 정리해줘"

| 단계 | 기존 (대화형) | Computer Use |
|---|---|---|
| 1 | 내가 웹사이트 방문 | AI가 직접 브라우저 열기 |
| 2 | 내가 스크린샷/텍스트 복사 | AI가 화면을 보고 정보 파악 |
| 3 | 복사한 내용을 AI에게 붙여넣기 | — (이미 보고 있음) |
| 4 | AI가 텍스트로 정리해서 답변 | AI가 직접 엑셀/문서에 표 입력 |
| 5 | 내가 다시 복사해서 문서에 옮기기 | — (이미 입력 완료) |

> 💡 차이점: 기존에는 **나 → AI → 나** 왕복이 필요했는데, Computer Use는 **AI가 처음부터 끝까지** 해요.

---

## 현재 할 수 있는 것 / 아직 어려운 것

| 할 수 있는 것 ✅ | 아직 어려운 것 ⏳ |
|---|---|
| 웹 브라우저에서 정보 검색·수집 | 복잡한 멀티스텝 업무 (10단계+) |
| 간단한 폼 작성·제출 | 보안이 필요한 금융 거래 |
| 스프레드시트에 데이터 입력 | 실시간 빠른 반응이 필요한 작업 |
| 파일 다운로드·정리 | 2단계 인증이 필요한 로그인 |
| 반복 클릭 작업 (데이터 입력) | 사람의 눈치·판단이 필요한 소통 |
| 스크린샷 찍고 내용 분석 | 예외 상황 대처 (팝업, 에러) |

## 주의사항 — 내 지팡이를 빌려주는 거예요

| ⚠️ 주의 | 이유 |
|---|---|
| 비밀번호·인증 정보 주의 | AI가 화면을 보므로 민감 정보 노출 가능 |
| 중요 작업은 감시 모드로 | 결제·삭제 같은 되돌릴 수 없는 작업은 반드시 지켜보기 |
| 결과 검증 필수 | AI가 잘못 클릭하거나 엉뚱한 곳에 입력할 수 있음 |
| 네트워크 환경 확인 | 느린 네트워크에서는 화면 인식이 부정확해질 수 있음 |

> 💡 비유: 다른 마법사에게 내 지팡이를 빌려주는 거예요. 실력 있는 마법사지만, **내 지팡이의 버릇(환경)**을 100% 알진 못해요. 옆에서 지켜보세요.

## 미래 — 어디까지 갈까?

| 시기 | 예상 |
