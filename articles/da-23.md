# 주문 교실 #23 — Claude의 6가지 얼굴 🔮

> 스위스 아미 나이프(Swiss Army Knife)를 사셨어요.
> 그런데 병따개만 쓰고 있었다면? 나이프, 가위, 드라이버, 톱, 줄이 더 있는데요.
> Claude도 마찬가지예요. **채팅만 쓰면 1/6만 쓰는 거예요.**

---

## 오늘 배울 것 — 한눈에

| 순서 | 얼굴 | 비유 |
|---|---|---|
| 1 | 채팅 (Chat) | 대표번호 상담원 |
| 2 | Cowork (데스크톱 앱) | 실제로 문서 쓰는 워드 화면 |
| 3 | Claude Code (터미널) | 말로 시키면 코딩해주는 개발자 |
| 4 | Claude for Excel | 엑셀 안에 앉은 마법사 |
| 5 | 프로젝트 (Projects) | 단톡방+드라이브+노트를 한 곳에 |
| 6 | 스킬 (Skills) | 직무기술서가 붙은 전문 직원 |

---

## 6가지 얼굴 비교표

| 얼굴 | 뭘 하는지 | 비유 | 언제 쓰는지 |
|---|---|---|---|
| **채팅** | 질문하고 답 받기 | 대표번호로 전화하면 받는 상담원 | 빠른 질문, 아이디어 정리 |
| **Cowork** | 내 파일을 AI가 직접 열고 수정 | 회의 끝나고 실제로 문서 쓰는 워드 화면 | PDF 30개 요약, 엑셀 비교분석 |
| **Claude Code** | 코드 작성·실행·디버깅 | 말로 시키면 코딩해주는 개발자 | 바이브코딩, 자동화 구축 |
| **Claude for Excel** | 엑셀 안에서 피벗·차트·수식 조작 | 엑셀 안에 앉은 마법사 | 피벗테이블, 차트, 복잡한 수식 |
| **프로젝트** | 폴더별로 맥락 분리 | 단톡방+드라이브+노트를 한 곳에 | 사건별·고객별 업무 분리 |
| **스킬** | 반복 주문을 명령어 하나로 | 직무기술서(JD)가 붙은 전문 직원 | 매번 같은 작업 반복할 때 |

> 💡 스위스 아미 나이프 비유: 채팅은 병따개예요. 가장 많이 쓰지만, 나이프(Cowork)로 잘라야 할 때, 가위(Code)로 오려야 할 때, 드라이버(Excel)로 조여야 할 때가 있어요. **도구를 골라 쓸 줄 아는 게 프로예요.**

---

## 1. 채팅 (Chat) — 대표번호 상담원

가장 기본이에요. claude.ai에 접속해서 질문하고 답 받기.
"이 계약서 조항이 뭔 뜻이야?", "이 증상에 맞는 약 뭐야?" — 이런 빠른 질문에 좋아요.

> 💡 비유: 회사 대표번호로 전화하면 받는 상담원이에요. 뭐든 물어볼 수 있지만, 내 서류를 직접 만져주지는 않아요. (웹 채팅 인터페이스)

---

## 2. Cowork (데스크톱 앱) — 실제로 문서 쓰는 워드 화면

Claude 데스크톱 앱을 열면, **내 컴퓨터 파일을 AI가 직접 열고, 수정하고, 정리해요.**
채팅이 "말로만 도와주는 상담원"이었다면, Cowork는 직접 문서를 써주는 비서예요.

> 💡 비유: 회의 끝나고 실제로 문서 쓰는 워드 화면이에요. 상담원이 아니라 실무자. (Claude Desktop Cowork 모드)

<details>
<summary>▸ 이런 상황에서 채팅 대신 Cowork</summary>

| 상황 | 채팅으로 하면 | Cowork로 하면 |
|---|---|---|
| PDF 30개 요약 | 하나씩 복붙해서 물어보기 | 파일 30개 드래그 → "요약해줘" |
| 워드 파일 수정 | 답변 복사해서 내가 붙여넣기 | AI가 직접 파일 열어서 수정 |
| 엑셀 비교분석 | 스크린샷 찍어서 물어보기 | 파일 두 개 넣고 "비교해줘" |

핵심: **내 파일을 건네줄 수 있느냐**가 채팅과 Cowork의 차이예요.

</details>

---

## 3. Claude Code (터미널) — 말로 시키면 코딩해주는 개발자

터미널에서 Claude Code를 실행하면, **코드를 작성하고, 실행하고, 디버깅까지** 해요.
"바이브코딩"이 바로 이거예요. 말로 설명하면 코드가 완성돼요.

> 💡 비유: 옆에 앉아서 내 말을 듣고 코딩해주는 개발자예요. (Claude Code CLI)

<details>
<summary>▸ 비개발자도 쓸 수 있나요?</summary>

네! DA#1~4에서 배운 그 도구예요. 코드를 몰라도 됩니다.

| 비개발자가 Claude Code로 하는 것 | 예시 |
|---|---|
| 파일 정리·분석 | "이 폴더의 PDF 전부 요약해서 표로 만들어줘" |
| 자동화 | "매주 월요일 보고서 양식 자동 생성해줘" |
| 웹사이트 만들기 | "내 포트폴리오 사이트 만들어줘" |

코딩을 하려고 쓰는 게 아니라, **코딩이 필요한 일을 말로 시키려고** 쓰는 거예요.

</details>

---

## 4. Claude for Excel — 엑셀 안에 앉은 마법사

엑셀을 열면 사이드 패널에 Claude가 있어요. **피벗테이블, 차트, 수식을 대화로 조작해요.**
DA#20에서 자세히 다뤘어요.

> 💡 비유: 스위스 아미 나이프의 드라이버처럼, 엑셀이라는 나사를 정확히 조이는 전용 도구예요. (Claude for Excel 애드인)

<details>
<summary>▸ 채팅에서 엑셀 질문하는 것과 뭐가 다른가요?</summary>

| | 채팅에서 엑셀 질문 | Claude for Excel |
|---|---|---|
| 수식 | "이런 수식 알려줘" → 내가 복붙 | AI가 셀에 직접 입력 |
| 피벗 | 만드는 법 설명 | 클릭 한 번에 생성 |
| 차트 | 이미지로 보여줌 | 엑셀 차트로 바로 삽입 |

> 💡 설명해주는 선생님 vs 직접 해주는 조교. (DA#20 참조)

</details>

---

## 5. 프로젝트 (Projects) — 단톡방+드라이브+노트를 한 곳에

CLAUDE.md(마법서) + 파일 + 대화를 **폴더 하나에 묶는 기능**이에요.
DA#12에서 자세히 다뤘어요.

> 💡 비유: 카톡 단톡방 + 구글 드라이브 + 메모장을 한 곳에 합친 거예요. 사건마다 방이 따로 있으니 맥락이 안 섞여요. (Project Instructions + Knowledge)

<details>
<summary>▸ 프로젝트 없이 쓰면 생기는 일</summary>

| 프로젝트 없이 | 프로젝트로 분리하면 |
|---|---|
| 세무 상담하다가 소송 맥락이 섞임 | 사건별로 깔끔하게 분리 |
| 매번 "나는 변호사고, 이 건은..." 반복 | 한 번 설정하면 자동 적용 |
| 파일을 매번 다시 첨부 | 프로젝트에 저장해두면 항상 참조 |

> 💡 서랍 없이 책상 위에 서류를 다 쌓아두는 것 vs 서랍별로 정리하는 것. (DA#12 참조)

</details>

---

## 6. 스킬 (Skills) — 직무기술서가 붙은 전문 직원

반복되는 주문을 저장해서 **명령어 하나로 실행**하는 기능이에요.
DA#13에서 자세히 다뤘어요.

> 💡 비유: 스위스 아미 나이프에 자기만의 도구를 추가하는 거예요. "계약서 리뷰해줘"라고 하면 늘 같은 체크리스트로 검토하는 전문 직원. 직무기술서(JD)가 붙어있어서 알아서 해요. (Custom Slash Commands)

<details>
<summary>▸ 매번 시키기 vs 스킬로 시키기</summary>

| | 매번 시키기 | 스킬로 시키기 |
|---|---|---|
| 주문 | "계약서 리뷰해줘. 갑을 관계 확인하고, 위약금 조항 체크하고, 해지 조건 확인하고..." | `/contract-review` |
| 품질 | 매번 빠뜨리는 항목 생김 | 체크리스트가 고정돼서 항상 일정 |
| 속도 | 매번 3분 주문 작성 | 3초 |

> 💡 주문서에 적어두면 한 마디로 실행. (DA#13 참조)

</details>

## 채팅만 쓸 때 vs 6가지 다 쓸 때

| 항목 | 채팅만 쓸 때 (Before) | 6가지 다 쓸 때 (After) |
|---|---|---|
| 파일 작업 | 복사-붙여넣기 반복 | Cowork가 직접 열고 수정 |
| 엑셀 | 수식 물어보고 내가 입력 | Claude for Excel이 직접 조작 |
| 반복 업무 | 매번 긴 주문 작성 | 스킬 명령어 한 줄 |
| 맥락 관리 | 매번 처음부터 설명 | 프로젝트로 사건별 분리 |
| 자동화 | 불가능 | Claude Code로 구축 |
| 체감 | "AI 좀 써봤는데 그저 그래요" | "이거 없으면 일 못 해요" |

> 💡 스위스 아미 나이프 6개 도구를 다 펼치면, 같은 나이프인데 할 수 있는 일이 완전히 달라져요.
